# R_basics

## Data Types

**numeric**
```R
# numeric
x <- 10.5
class(x)
x
```

```cml
[1] "numeric"
[1] 10.5
```

**integer**
```R
# integer
x <- 1000L
class(x)
x
```
```cml
[1] "integer"
[1] 1000
```

**complex**
```R
# complex
x <- 9i + 3
class(x)
x
```
```cml
[1] "complex"
[1] 3+9i
```

**character**
```R
# character
x <- "its a string"
class(x)
x
```
```cml
[1] "character"
[1] "its a string"
```

**logical**
```R
# logical
x <- TRUE
class(x)
x
```
```cml
[1] "logical"
[1] TRUE
```
